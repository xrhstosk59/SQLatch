## Εισαγωγή

Μία <span style="color:coral">
βάση δεδομένων
</span> είναι ένα εργαλείο για τη συλλογή και την οργάνωση πληροφοριών. Χρησιμοποιείται για την αποθήκευση, την ανάκτηση και τη διαχείριση δεδομένων, παρέχοντας αποτελεσματικό τρόπο οργάνωσης και πρόσβασης σε πληροφορίες.

<span style="color:coral">
Πίνακας
</span> είναι μία δομή δεδομένων που αποθηκεύει πληροφορίες σε μορφή πλέγματος. Κάθε πίνακας αναπαριστά ένα σύνολο δεδομένων (όπως μία κατηγορία πληροφοριών). Για παράδειγμα, ένας πίνακας "employees" μπορεί να περιέχει στοιχεία όπως το Όνομα, το Επώνυμο και το μισθό του κάθε υπαλλήλου.

![Screenshot](MDGuides/Lessons/Intro/images/first.png)

<span style="color:coral">
Στήλη
</span>
είναι μια μονάδα δεδομένων εντός ενός πίνακα. Κάθε στήλη ορίζει τον τύπο των δεδομένων που περιέχει. Για παράδειγμα, στον πίνακα "employees", μια στήλη "firstname" μπορεί να περιέχει ονόματα σε μορφή κειμένου, ενώ μια άλλη στήλη "salary" μπορεί να περιέχει τους μισθούς. Κάθε εγγραφή (ή γραμμή) στον πίνακα αντιστοιχεί σε ένα σύνολο δεδομένων για έναν υπάλληλο.
