## Ασκήσεις

**Άσκηση 1 (PRIMARY KEY)**

Δημιουργήστε έναν πίνακα με το όνομα Φοιτητές που να περιέχει τα εξής πεδία: ΑΜ (αριθμός μητρώου) ως PRIMARY KEY, Όνομα, Επώνυμο και ΗμερομηνίαΓέννησης.

<details>
  <summary>Δείξε την ενδεικτική λύση</summary>

```sql
CREATE TABLE Φοιτητές (
ΑΜ INT PRIMARY KEY,
Όνομα VARCHAR(25),
Επώνυμο VARCHAR(25),
ΗμερομηνίαΓέννησης DATE
);
```

**Άσκηση 2 (PRIMARY KEY)**

Δημιουργήστε έναν πίνακα με το όνομα Μαθήματα που να περιέχει τα εξής πεδία: ΚωδικόςΜαθήματος ως PRIMARY KEY, ΌνομαΜαθήματος και Εξάμηνο.

<details>
  <summary>Δείξε την ενδεικτική λύση</summary>

```sql
CREATE TABLE Μαθήματα (
ΚωδικόςΜαθήματος INT PRIMARY KEY,
ΌνομαΜαθήματος VARCHAR(60),
Εξάμηνο INT
);
```

</details>

**Άσκηση 3 (FOREIGN KEY)**

Δεδομένου ότι έχετε τους πίνακες Φοιτητές και Μαθήματα όπως παραπάνω, δημιουργήστε έναν πίνακα Εγγραφές όπου κάθε φοιτητής μπορεί να εγγραφεί σε πολλά μαθήματα. Ο πίνακας Εγγραφές θα περιέχει τα πεδία ΑΜ, ΚωδικόςΜαθήματος και θα έχει FOREIGN KEY συσχετισμούς με τους πίνακες Φοιτητές και Μαθήματα.

<details>
  <summary>Δείξε την ενδεικτική λύση</summary>

```sql
CREATE TABLE Εγγραφές (
ΑΜ INT,
ΚωδικόςΜαθήματος INT,
PRIMARY KEY (ΑΜ, ΚωδικόςΜαθήματος),
FOREIGN KEY (ΑΜ) REFERENCES Φοιτητές(ΑΜ),
FOREIGN KEY (ΚωδικόςΜαθήματος) REFERENCES Μαθήματα(ΚωδικόςΜαθήματος)
);
```

</details>

**Άσκηση 4 (FOREIGN KEY)**

Δημιουργήστε έναν πίνακα με το όνομα Καθηγητές που να περιέχει τα εξής πεδία ID_Καθηγητή ως PRIMARY KEY, Όνομα, Επώνυμο, ΚωδικόςΜαθήματος και FOREIGN KEY συσχετισμό με τον πίνακα Μαθήματα.

<details>
  <summary>Δείξε την ενδεικτική λύση</summary>

```sql
CREATE TABLE Καθηγητές (
ID_Καθηγητή INT PRIMARY KEY,
Όνομα VARCHAR(255),
Επώνυμο VARCHAR(255),
ΚωδικόςΜαθήματος INT,
FOREIGN KEY (ΚωδικόςΜαθήματος) REFERENCES Μαθήματα(ΚωδικόςΜαθήματος)
);
```

</details>
